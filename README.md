## Let’s build it! (Create)

### Prompt
```text
Create a picture of a laptop device. It should be photo realistic.
```

### Prompt Coach Advise
```text
Generate a photo-realistic image of a Microsoft Surface laptop placed on a modern office desk.
The laptop should be open, displaying a clean desktop screen with subtle reflections on the
screen and keyboard. The desk environment should include minimalistic accessories like a wireless
mouse, a coffee mug, and a notepad. The background should suggest a professional workspace with
soft natural lighting coming from a nearby window. The overall tone should be clean, modern,
and realistic—suitable for a corporate presentation demo.
```
